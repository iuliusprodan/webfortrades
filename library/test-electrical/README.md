# Hartley Electrical - design library entry

Reference implementation for WebForTrades **quiet-premium-editorial** electric trade sites.

## When to reuse
- Domestic electricians, finish trades, premium sole traders
- Cambridge / UK local service businesses

## Key patterns
1. **Sticky condensing header** with phone CTA always visible
2. **Hero focal frame** (dark gradient + caption) when real photos unavailable
3. **Scroll reveal** on sections, stagger on cards
4. **FAQ accordion** with chevron rotation
5. No em dashes in copy
